Users.json - файл, в который программа записывает отсортированных пользователей, вызывается после прочтения и каждый
раз, когда меняется список. Этот же файл читается при запуске.

Все аналогично для MessagesContoller.

Если пользователей меньше 2, то post не будет создавать сообщения.
(Но сообщения могут все еще хранится, если они были прочитанны из файла)
